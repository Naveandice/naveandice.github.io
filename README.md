# naveandice.github.io

This repository contains the source code of <https://naveandice.github.io>.

Feel free to fork this repo and change all contents to yours. You can even port
this website to Jekyll or whatever static site generator you like.

Shout out to Simon Willison and their [Today I Learned][til1] website where
I learned that [Jekyll is not required to run a website on GitHub Pages][til2].
